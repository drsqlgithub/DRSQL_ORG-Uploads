Code in this directory is designed to be in a schema named Tools. If you need to change to something else, just replace Tools with your desired schema.

Each file also includes a create for the Tools schema

Effective Security - returns the things that the current user in context can do

Simple Date Dimension Create And Load - Creates a table for a date dimension that you can use for date math/grouping

Simple Time Dimension Create And Load - Creates a table for a time dimension that you can use for date math/grouping
